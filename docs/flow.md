### Flow 

- rename hook files
- change mode
- parse config (yaml or json)
- append arguments at the end 


### Tools

- Bazel testing
- cmake builds
- .clang-format for formatting (using clang)
- CMakeLists.txt for cmake
- appveyor.yml for ci with bazel
- .editorconfig
